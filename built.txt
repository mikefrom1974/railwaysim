built CA PKI
    checks for existing root CA, generates if missing
    clients must have a valid token to get a cert (reality = TPM, etc)

built simulated train
    spawned via API call
    registers for its own certificate with issue token
    connects to RabbitMQ to listen for commands
    sends telemetry every second(ish) to API Gateway (Kong)

built RabbitMQ
    RabbitMQ container pulls server certificate from PKI.
    Trains register with RabbitMQ and subscribe to queue.

built kafka
    Kafka container pulls server certificate from PKI.
    Kafka will receive telemetry from trains via API Gateway.
    Redis / Prometheus will have telemetry pulled from Kafka via consumer.

built Kong API Gateway
    Trains push telemetry to kong, which forwards to the custom go injecter.
    (the built in kong log plugin was stuck on version 0 for the community build)
    The injecter will then forward to the kafka service.

built Injector
    Pulls telemetry from Kong, and pushes it to Kafka.

built Redis
    Stores current telemetry for each train
    Will be polled by SPOG UI

built BFF
    BackEnd-For-FrontEnd to act as a go-between from SPOG to redis and rabbit.

built SPOG
    React UI to pull train list from redis and show info/control widgets

built Prometheus
    TSDB that scrapes the exporter /metrics and feeds Grafana

built Grafana
    Dashboard for historical Prometheus data
